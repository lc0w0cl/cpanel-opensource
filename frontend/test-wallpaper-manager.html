壁纸管理器功能测试

测试重构后的主题设置功能

📸 当前壁纸预览
显示当前使用的壁纸，支持实时预览模糊度和遮罩效果。 点击预览按钮可全屏查看效果。
🎛️ 实时控制
通过滑动条实时调整模糊度（0-20px）和遮罩透明度（0-80%）， 所有调整都会实时反映在预览中。
🖼️ 壁纸库
展示所有历史上传的壁纸，支持网格布局显示。 每张壁纸显示缩略图、文件名和大小信息。
➕ 添加壁纸
点击"添加壁纸"按钮上传新的背景图片， 支持 JPG、PNG 等常见格式，自动保存到 backgrounds 目录。
✅ 应用壁纸
点击壁纸缩略图上的应用按钮， 即可将该壁纸设置为当前背景，当前壁纸会显示特殊标识。
🗑️ 删除壁纸
点击删除按钮可移除不需要的壁纸文件， 当前正在使用的壁纸无法删除，确保系统稳定性。

🔧 API 接口测试

获取历史壁纸列表

GET /api/system-config/wallpaper/history

应用指定壁纸

POST /api/system-config/wallpaper/{wallpaperId}/apply

删除指定壁纸

DELETE /api/system-config/wallpaper/{wallpaperId}

📖 使用说明

  1. 查看当前壁纸：在预览区域查看当前壁纸效果，点击预览按钮全屏查看
  2. 调整效果：使用滑动条实时调整模糊度和遮罩透明度
  3. 上传新壁纸：点击"添加壁纸"按钮选择图片文件上传
  4. 选择壁纸：在壁纸库中点击任意壁纸缩略图进行选择
  5. 应用壁纸：点击壁纸上的应用按钮将其设为当前背景
  6. 删除壁纸：点击删除按钮移除不需要的壁纸（当前壁纸除外）
  7. 保存设置：点击"保存设置"按钮将当前配置保存到数据库
  8. 重置默认：点击"重置默认"按钮恢复到系统默认壁纸
